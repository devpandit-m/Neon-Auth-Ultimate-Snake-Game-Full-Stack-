# 🚀 Neon Auth & Ultimate Snake Game (Full-Stack)

A seamless integration of a modern **Web UI** and a **Python-based Game Engine** using **Flask**.

## 🌟 Features
- **Neon Animated UI:** Built with pure HTML5/CSS3 (Keyframe animations).
- **Secure Backend:** Flask server handling user validation and session logic.
- **Dynamic Integration:** Uses Python's `subprocess` to launch a Pygame instance from the web trigger.
- **Ultimate Snake Game:** Featuring emoji food, score tracking, and speed scaling.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Animations), JavaScript (Fetch API)
- **Backend:** Python, Flask
- **Game Engine:** Pygame
- **Storage:** LocalStorage (for client-side persistence)
